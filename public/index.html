GET un vin par nom /nom/:nom

Renvoie un vin en fonction de son nom.

GET /nom/:nom

Exemple de chemin

https://wine-api-one.vercel.app/nom/chateau-margaux

Exemple de réponse :

      {
        "vin": {
          "nom": "Château Margaux",
          "region": "Bordeaux",
          "cepage": "Cabernet Sauvignon",
          "type": "Rouge",
          "annee": 2015,
          "note": 95,
          "description": "Un vin rouge d'exception de la région de Bordeaux...",
          "recommandation": "Parfait pour accompagner un filet mignon ou un plateau de fromages.",
          "image": "lien_vers_image1.jpg"
        }
      }
